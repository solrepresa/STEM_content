# Material de cierre de unidad — Introducción a las Ciencias de Datos

HTMLs de cierre de unidad para una materia universitaria de primer año. Cada archivo es autocontenido: abrís en el navegador y funciona, sin servidor, sin dependencias, sin instalación.



## Qué es esto

Material complementario para el curso *Introducción a las Ciencias de Datos* (UCA, 2026), diseñado para el cierre de cada unidad temática.

No son resúmenes ni diapositivas. Son piezas editoriales interactivas que buscan conectar el contenido técnico con preguntas que tienen sentido fuera del aula: con la familia, con amigos, con gente de otras carreras.

El enfoque no es especialista sino integrador. Cada unidad cruza ciencia de datos con ética, filosofía, biología, arte o urbanismo, según el tema.



## Cómo usarlo

Descargá el HTML que te interese y abrilo en cualquier navegador moderno. 
No necesita internet después de cargado (las fuentes de Google Fonts se cachean).



## Feedback

Este material lo usé con mis estudiantes, pero necesito perspectivas externas. Si lo probás en tu aula, si algo no funciona, si algo te parece que podría ser mejor, abrí un Issue o escribime directamente. Cualquier tipo de feedback es bienvenido.
